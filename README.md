# Week 1 - Structures in C++

## Objectives

* Learn Structure Declaration
* Initialize Structure Variables
* Store Student, Date and Employee Information
* Access Structure Members

## Experiments

### Experiment 1

Create a Student structure containing:

* Student ID
* Student Name
* Course Code
* Course Name
* Course Marks

### Experiment 2

Create a Date structure and store a date entered by the user.

### Experiment 3

Create an Employee structure and display information of three employees using a separate display function.

### Experiment 4

Create multiple Student objects and calculate total marks.

## Concepts Used

* Structures
* User Input
* Functions
* Data Storage

## Language

C++
